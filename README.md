# Roomie Ecosystem

A modern, minimal, and lightning-fast landing page introducing the Roomie application suite. Built with [Astro](https://astro.build/).

## Ecosystem Applications

- **roomieVerse**: Your digital universe for co-living. Manage tasks, events, and communication seamlessly with your roommates.
- **roomio**: The smartest way to handle shared expenses and bills. No more awkward conversations about who owes what.
- **roomiBase**: Find the perfect room or roommate. Advanced matchmaking based on lifestyle, habits, and preferences.

## Tech Stack

- **Framework**: Astro (Zero JS by default for fast load times)
- **Styling**: Vanilla CSS, Flexbox/CSS Grid
- **Design Language**: Clean UI, Glassmorphism panels, Smooth hover micro-animations
- **Typography**: [Outfit](https://fonts.google.com/specimen/Outfit) (Google Fonts)

## Getting Started

To run this project locally, make sure you have Node.js installed, then follow these steps:

1. **Install dependencies**
   ```bash
   npm install
   ```

2. **Start the development server**
   ```bash
   npm run dev
   ```

3. **View in browser**
   Open your browser and navigate to `http://localhost:4321` to see the landing page in action.

## Features

- **Fully Responsive**: Adapts beautifully to mobile, tablet, and desktop screens.
- **Premium Aesthetics**: Features a clean white background, soft shadow glassmorphism, and balanced visual sizing for disparate logo formats (tall vs wordmark).
- **SEO Ready**: Semantic HTML structure optimized for search engines.
